Database Beheer

Lokale beheerpagina voor de gedeelde productdatabase.

Deze pagina helpt je om een nieuwe gedeelde products.json te bouwen voor de hele app. De hoofdflow is nu: NEVO CSV + OFF JSON samenvoegen tot een nieuwe database.

Bronbestand: public/products.json Output: nieuwe products.json Na afloop: npm run build en deploy

Werkwijze

  1. Zet je NEVO Excel eerst om naar CSV, of gebruik direct je bestaande NEVO2025_v9.0.csv.
  2. Kies hieronder je NEVO-bestand en je off_nl.json.
  3. Klik op Bouw products.json.
  4. Controleer de samenvatting.
  5. Download daarna de nieuwe products.json.
  6. Vervang lokaal public/products.json, draai npm run build en deploy.

Build Uit Bronnen

Gebruik hier je twee bronbestanden. NEVO levert de officiële basisproducten, OFF levert de merk- en supermarktproducten.

Ondersteunt jouw pipe-gescheiden NEVO export zoals NEVO2025_v9.0.csv.
Verwacht het formaat met { "meta": ..., "producten": [...] }.
